# Expo + OXC

This is an [Expo](https://expo.dev) project created with [`create-expo-app`](https://www.npmjs.com/package/create-expo-app).

You may refer to the original [README.md](./docs/EXPO_README.md) generated by `create-expo-app`.

---

This is a project template for an Expo app which uses [oxc](https://oxc.rs/) to lint and format files.

## Get Started

1. Install dependencies

```bash
pnpm install
```

2. Start the app

```bash
pnpm start
```

## Scripts

- `pnpm start`
- `pnpm android`
- `pnpm ios`
- `pnpm web`
- `pnpm ox:lint` - Checks for linting errors
- `pnpm ox:lint:fix` - Fixes linting errors
- `pnpm ox:format:check` - Checks for oxfmt errors
- `pnpm ox:format` - Applies oxfmt formatting

## Recommended Extensions

Recommended extensions are provided in the `.vscode/extensions.json`.
